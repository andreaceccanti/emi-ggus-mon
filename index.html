EMI GGUS Monitor

The EMI project provides a software platform for high performance distributed computing which forms the core of grid middleware distributions used by scientific research communities and distributed computing infrastructures all over the world.

The EMI GGUS support monitor is a python script that provides insight on the status and performance of EMI support units leveraging the GGUS ticketing system web services.

Dependencies

  • python >= 2.6
  • python-suds >= 0.3.9

Installation instructions

To install this package on your machine, please type (as root):

# python setup.py install

Usage

$ ggus.py

will generate the support status report, be patient.

$ ggus.py --help

provides insights about other options.
